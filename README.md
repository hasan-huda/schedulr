# Schedulr

A full-stack scheduling application built with Next.js, Prisma, and PostgreSQL.
Includes authentication, role-based access control (RBAC), and multi-tenant workspaces.

## Tech Stack
- Next.js (App Router)
- TypeScript
- Prisma
- PostgreSQL
- NextAuth (Auth.js)
- Tailwind CSS

## Features
- OAuth authentication
- Per-user demo workspace
- RBAC (Admin / Member)
- Secure server actions
- Database-backed sessions

## Getting Started

```bash
npm install
npm run dev
